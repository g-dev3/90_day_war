## HTML & CSS

### What is HTML?

HTML stands for HyperText Markup Language. It is the fundamental tool we use to create websites. HTML provides the structure and content of web pages, allowing us to define elements like headings, paragraphs, links, images, and more that browsers can interpret and display to users.

### What is HTML Element?

(`<button>`) Element is a generic term that we can use to describe anything that we can display on webpage.

### What is Syntax?

rules of writing code. 
Example -

### What is HTML Tag?

An HTML tag is a keyword surrounded by angle brackets `< >` that tells the browser how to display content. Tags come in pairs - an opening tag (e.g., `<p>`) and a closing tag (e.g., `</p>`). The opening tag marks the beginning of an element, and the closing tag marks the end. Some tags are self-closing, like `<img/>` or `<br/>`. Tags define what type of content they contain, such as `<h1>` for headings, `<p>` for paragraphs, or `<a>` for links.

### What is Anchor Element?

An anchor element is created using the `<a>` tag and is used to create hyperlinks on web pages. The `href` attribute is used to specify the URL or destination that the link should point to. Example: `<a href="https://example.com">Click here</a>`. The anchor element allows users to navigate to different pages, websites, or sections within the same page. The text between the opening `<a>` and closing `</a>` tags is what appears as the clickable link to the user.

---

**Note:** In HTML, extra spaces, tabs, and line breaks (whitespace) are ignored by the browser. When you have multiple consecutive spaces in your HTML code, they will be treated as a single space, and extra blank lines will be collapsed into one. The browser only displays visible content and renders whitespace according to the CSS styling and the nature of the elements. This means you can format your HTML code with multiple spaces and line breaks for readability, and it won't affect how the page appears to the user.
